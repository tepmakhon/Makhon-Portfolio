## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/tepmakhon/Makhon-Portfolio.git
```

Go to the project

```bash
cd Makhon-Portfolio
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Build production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
